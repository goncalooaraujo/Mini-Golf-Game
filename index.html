Desenvolvimento

Seu Nome - Desenvolvedor Principal

Este jogo foi desenvolvido como um projeto de aprendizado e demonstração de habilidades em desenvolvimento de jogos usando tecnologias web modernas.

Tecnologias Utilizadas

  • Three.js - Motor de renderização 3D
  • Cannon.js - Motor de física
  • HTML5 & CSS3 - Interface e estilização
  • JavaScript - Programação e lógica do jogo

Recursos

Modelos 3D, texturas e sons utilizados neste jogo foram obtidos de diversas fontes, incluindo:

  • Sketchfab - Modelos 3D
  • Freesound - Efeitos sonoros
  • Pixabay - Músicas e texturas
